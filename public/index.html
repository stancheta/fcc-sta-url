STA URL

Get your "shortened" URL

This is a clone of TinyURL. Below, you can input a url and a "shortened" version will be returned. Otherwise, you can find the instructions for the API below.

API instructions

Example Creation Usage

https://sta-url.herokuapp.com/new/https://www.google.com https://sta-url.herokuapp.com/new/http://steventancheta.com

Example Creation Output

{ "original_url":"https://www.google.com", "short_url":"https://sta-url.herokuapp.com/8170" }

Example Usage

https://sta-url.herokuapp.com/8170

will redirect to

https://www.google.com
